# GUI Flash Cards for Learning Languages - French
### Made with Tkinter
### A french word is displayed, the user has 3 seconds to figure out the english substitute for it
### The user has a choice of two buttons, check for knowing the word, cross for not
### Every check clicked will remove the current word from the wordbank of unknown words
### Progress is saved to csv via pandas
### The unknown list is loaded upon starting the program
### The goal is to learn a new language!
